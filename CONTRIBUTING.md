# Contributing

Thanks for your interest in contributing!

## Getting Started

1. Fork the repo
2. Clone your fork
3. Run `bun install`
4. Make your changes
5. Submit a pull request

## What We Need

- Bug fixes
- New features
- Documentation improvements
- Translations

## Code Style

- TypeScript
- Follow existing patterns
- Keep it simple

## Questions?

Open an issue or discussion.
